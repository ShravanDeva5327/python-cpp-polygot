# Polyglot Program

This clever little code is designed to be a chameleonâ€”it behaves as both a C++ program and a Python script. Depending on how you run it, you'll either see a "Hello from C++" message or a "Hello from Python" message.

### cpp:
```cpp

""""*/ <stdio.h>

int main() {
    printf("Hello from C++\n");
    return 0;
} /*
"""#*//*
print("Hello from Python") #*/
```
